# メトリクス

このページでは、評価に使用する定量的なメトリクスについて説明します。

## 定量メトリクス

### コード品質メトリクス

| メトリクス | 説明 | 測定方法 |
|-----------|------|---------|
| Lines of Code (LOC) | コード行数 | 自動計測 |
| Cyclomatic Complexity | 循環的複雑度 | 静的解析 |
| Test Coverage | テストカバレッジ | テスト実行 |
| Lint Errors | リントエラー数 | リンター |

### ドキュメント品質メトリクス

| メトリクス | 説明 | 測定方法 |
|-----------|------|---------|
| Word Count | 単語数 | 自動計測 |
| Reading Level | 読解レベル | Flesch-Kincaid |
| Link Validity | リンクの有効性 | リンクチェッカー |
| Structure Score | 構造スコア | カスタム評価 |

## 計算式

### 総合スコア

```
総合スコア = Σ(カテゴリスコア × 重み) / Σ重み
```

### カテゴリ別重み

| カテゴリ | デフォルト重み |
|---------|--------------|
| 正確性 | 0.30 |
| 完全性 | 0.25 |
| 可読性 | 0.20 |
| 一貫性 | 0.15 |
| 保守性 | 0.10 |

## メトリクス収集

### 自動収集

以下のメトリクスは自動的に収集されます：

- コード行数
- 複雑度
- リントエラー
- 単語数

### 手動評価

以下の項目は手動で評価されます：

- 正確性（レビュアーによる確認）
- 可読性スコア
- ドキュメント構造
