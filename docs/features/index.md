# 機能

Coding Agent Comparison の主な機能について説明します。

## コア機能

### エージェント比較

複数の AI コーディングエージェントを同一の基準で評価・比較します。

- **公平な比較**: 同じタスクを各エージェントに実行させ、結果を比較
- **定量的評価**: 明確なメトリクスに基づいた評価
- **定性的評価**: ドキュメントの読みやすさ、構造などの評価

### ドキュメント生成評価

エージェントが生成するドキュメントの品質を多角的に評価します。

| 評価項目 | 説明 |
|---------|------|
| 正確性 | 技術的に正確な内容か |
| 完全性 | 必要な情報が網羅されているか |
| 可読性 | 読みやすく理解しやすいか |
| 構造性 | 論理的な構造になっているか |

### 再現可能な評価

同じ入力に対して同じ結果が得られる再現可能な評価プロセスを提供します。

```
入力 → エージェント実行 → 出力収集 → 評価 → レポート
```

## 拡張機能

### カスタムテストケース

独自のテストケースを追加して、特定のユースケースを評価できます。

### レポート生成

評価結果を自動的にレポートとして出力します。

- Markdown 形式
- HTML 形式
- JSON 形式（プログラム処理用）
