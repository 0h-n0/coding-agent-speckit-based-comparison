# 用語集

Coding Agent Comparison プロジェクトで使用される用語の定義です。

## A

### Agent（エージェント）

AI モデルを使用してタスクを実行するソフトウェアコンポーネント。
このプロジェクトでは、コーディングタスクを実行する AI を指します。

## B

### Benchmark（ベンチマーク）

エージェントの性能を測定するための標準化されたテスト。

## C

### Codex

OpenAI が提供するコード生成に特化した AI モデル。

### Claude

Anthropic が提供する会話型 AI アシスタント。

## E

### Evaluation（評価）

定められた基準に基づいてエージェントの出力を査定するプロセス。

## G

### Gemini

Google が提供するマルチモーダル AI モデル。

### GitHub Pages

GitHub が提供する静的サイトホスティングサービス。

## M

### Material for MkDocs

MkDocs 用の高機能テーマ。

### Metrics（メトリクス）

評価に使用される定量的な指標。

### MkDocs

Markdown ベースの静的サイトジェネレーター。

## R

### Reproducibility（再現性）

同じ入力に対して同じ結果が得られる性質。

## S

### Submodule（サブモジュール）

Git リポジトリ内の別の Git リポジトリへの参照。

### Speckit

仕様駆動開発をサポートするツールセット。

## T

### Test Case（テストケース）

エージェントを評価するための具体的なタスク定義。

### Transparency（透明性）

評価プロセスと基準が公開され、検証可能である性質。
