# Contracts

This feature (MkDocs GitHub Pages) does not require API contracts.

MkDocs generates a static documentation site with no backend API.
The "contracts" for this feature are the file structure and configuration schemas
defined in `data-model.md`.

## Relevant Schemas

See `data-model.md` for:
- Document page structure
- Navigation configuration
- mkdocs.yml schema
- GitHub Actions workflow schema
